# JobFair - Your easiest job finder online.

## Introduction

A job boarding website consist of job listing for jobseekers and jobseekers list for employers.

## Code Samples

Code functionalities are found on server/functions.php

## Installation

<b>Local installion </b> - create new database on phpmyadmin and name it jobfair_db then import the database file on database/jobfair_db.sql. After creating database, then you can go to the url localhost/jobfair/index.php<br>
<b>Online installation</b> - have your own domain, add the database file on the server and go to the url.
